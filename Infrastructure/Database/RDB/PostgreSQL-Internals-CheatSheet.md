# PostgreSQL Internals CheatSheet

PostgreSQL 是著名的开源关系型数据库，其内部由多个独立协作的子系统构成；了解其内部原理对于 DBA 以及开发人员都有着极大的意义，本文也即希望对于 PostgreSQL 中子系统的工作原理进行概述，给予读者高层次的理解。
